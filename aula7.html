Circuitos Digitais: Introdução à Linguagem C++ (C#)

Circuitos Digitais

  • Entrada, Processamento e Saída: Os circuitos digitais são compostos por três partes principais: entrada, processamento e saída.
  • Portas Lógicas: AND, OR, NOT. Estudo das tabelas verdade e circuitos elétricos com chaves, fontes, resistores e LEDs.
  • ICs: IC7408, IC7432, IC7404. Equações e diagramas das portas lógicas.

Introdução à Linguagem C#

  • Criação de um circuito com LED, resistor, protoboard e Arduino.
  • Divisão do código em 3 partes:
    1. Declaração de variáveis e constantes.
    2. Configuração (entradas, saídas, lógica e matemática).
    3. Repetições.

Conforme o código:


/*
 * Código para acender um LED com Arduino
 */

// Parte 1 - Declaração de variáveis e constantes
int ledPin = 13; // Define o pino do LED

// Parte 2 - Configuração
void setup() {
  pinMode(ledPin, OUTPUT); // Define o pino do LED como saída
}

// Parte 3 - Repetições
void loop() {
  digitalWrite(ledPin, HIGH); // Liga o LED
  delay(1000); // Espera 1 segundo
  digitalWrite(ledPin, LOW); // Desliga o LED
  delay(1000); // Espera 1 segundo
}
}

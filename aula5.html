Semáforo

Um semáforo feito com Arduino é um projeto simples que pode ser realizado virtualmente no Tinkercad ou fisicamente. O objetivo é conectar LEDs ao Arduino e controlá-los com um código que alterna entre os núcleos vermelho, amarelo e verde, simulando um semáforo comum de trânsito.

Montagem do Circuito

Para montar o circuito, é necessário um conjunto de componentes, incluindo:

  • Arduino Uno
  • Resistores de 220 Ω (6)
  • Chave Táctil Push- botão
  • LEDs Vermelho, Amarelo e Verde
  • Protoboard
  • Jumpers

Programação

A programação é feita utilizando a linguagem de programação do Arduino. O código é dividido em dois blocos principais:

  • Setup: Define as portas do Arduino como entrada ou saída e configura as variáveis.
  • Loop: Contém a lógica do semáforo, que alterna entre os núcleos vermelho, amarelo e verde, simulando o funcionamento de um semáforo.

Simulação no Tinkercad

O Tinkercad é uma plataforma online que permite criar modelos 3D, simular circuitos eletrônicos e programar interações de forma visual. Com o Tinkercad, é possível simular o funcionamento do semáforo antes de montá-lo fisicamente.

Função do Semáforo

O semáforo é um sistema de sinalização que controla o trânsito em cruzamentos de vias . Ele alterna entre as cores vermelho, amarelo e verde para indicar quando os carros podem avançar ou parar. O semáforo é uma ferramenta importante para garantir a segurança no trânsito.

Benefícios do Projeto

Este projeto é uma maneira prática e divertida de aprender eletrônica e programação com Arduino. Ele permite que os usuários experimentem e aprendam sobre o funcionamento de um semáforo, além de desenvolver habilidades em programação e eletrônica.

Ele consiste em conectar LEDs ao Arduino e controlá-los com um código que alterna entre os núcleos vermelho, amarelo e verde, simulando um semáforo comum de trânsito.
